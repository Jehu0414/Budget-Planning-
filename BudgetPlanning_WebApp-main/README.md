#Julie simple budget planner wep/mobile app
A simple, cross-device web/mobile application to track income, expenses, and net balance by month and year.
 
Key Features
 
- Filter transactions by month and year

- View real-time summary of total income, expenses, and net balance

- Add new transactions with description, amount, and type (income/expense)

- View all recorded transactions in a structured table format

- Responsive design optimized for web and mobile devices
 
Tech Stack
 
- HTML5 for structure

- CSS3 for styling

- JavaScript for interactivity and data management
 
How to Use
 
1. Clone or download the repository to your local machine

2. Open the  index.html  file in any modern web browser (desktop or mobile)

3. Use the month/year filter to view data for specific periods

4. Fill out the transaction form to add new income or expense entries

5. Click "View All" to see all recorded transactions
